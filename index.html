▶️ 에이전트 실행 방법

  • VS Code 실행 및 새 폴더 생성

  • 📝 프롬프트 분석 및 계획 수립

  • ⚙️ 코드 작성 및 수정 (실시간)
  • 🔄 필요시 터미널 명령 실행
  • 🔍 오류 감지 및 수정 (자동 디버깅)
  • 💬 코드 설명 및 추가 요청 대응

✏️ 효과적인 프롬프트 작성법

❌ 나쁜 프롬프트

"로또 번호 생성기 웹사이트 만들어줘"

✅ 좋은 프롬프트

"HTML, CSS, JavaScript로 구성된 로또 번호 추첨기 웹사이트를 만들어주세요. 다크모드 레이아웃에 원형 UI로 1~45번 공이 물리엔진 기반으로 움직이며, 단일 조합과 5개 조합 모드를 선택할 수 있게 해주세요. 번호별로 색상도 다르게 표시해주세요."

📝 프롬프트 작성 5가지 원칙

  • 💭 구체적으로 작성하기: 원하는 기능, 디자인, 레이아웃을 상세히 설명
  • 🔍 참고 사이트 제공하기: 비슷한 예시 사이트 URL 첨부
  • 📂 파일 구조 명시하기: HTML, CSS, JS 등 필요한 파일 목록
  • ✅ 필수 구현 항목 나열하기: 꼭 포함되어야 할 기능들
  • 🎨 세부 스타일 지정하기: 색상, 폰트, 애니메이션 등

📋 로또 번호 추첨기 프롬프트 예시

📝 개조식 최종 프롬프트

🔰 전체 프롬프트

HTML, CSS, JavaScript로 구성된 웹사이트를 만들어주세요.

파일 구성:
- index.html : 전체 화면 구성
- styles.css : 스타일링 및 애니메이션 처리
- script.js : 모든 기능 동작 처리, 이벤트 관리, 상태 관리

필수 구현 항목:
디자인 / UI
- 다크모드 레이아웃 (#121212)
- 상단 로고 이미지 + 사이트명
- 단일조합 / 5조합 모드 선택 라디오 버튼
- 로또 머신 원형 UI (1~45번 공 배치)
- 컨트롤 패널 : 추첨하기, 결과 바로보기, 속도변경, 사운드 토글 버튼
- 결과영역 : 추첨된 공 표시 + 저장하기 버튼
- 저장된 번호 모달
- 당첨확인 모달 (동행복권 크롤링 방식)

공 움직임 및 애니메이션 처리
- 추첨 시작 시 머신 내부 공들이 자연스럽게 에어효과 및 물리엔진 기반으로 움직임
- 공끼리 충돌 / 무작위 위치 이동 / 자연스러운 개별 공 움직임 처리
- 로또 추첨기 스타일 팬 효과 적용
- 원통 안에서 공기의 저항을 받는 느낌으로 공들이 자유롭게 분산 이동하며 회전, 충돌 처리

공 추첨 애니메이션
- 공 추첨 후 해당 공은 바로 머신 하단 구멍으로 자연스럽게 떨어지는 애니메이션 처리
- 공 크기 축소 / 투명도 하락 / 자연스러운 낙하 및 사라짐
- 추첨된 공은 떨어지자마자 실시간으로 결과영역에 오름차순 정렬 후 바로 표시

번호별 볼 색상 지정
- 1~10 : 노랑
- 11~20 : 파랑
- 21~30 : 빨강
- 31~40 : 회색
- 41~45 : 초록

5조합 모드 처리 (핵심 로직)
- 각 조합의 번호는 중복 가능
- 단, "조합 전체"가 이전 조합과 완전히 동일하면 안 됨
- 이전 조합들과 숫자 6개 전부 동일하면 다시 뽑기
- Set 또는 JSON.stringify 비교 방식 사용
- 각 조합은 조합별로 묶어서 가로정렬 처리
- 각 조합 오른쪽 끝에는 '저장하기' 버튼 배치
- 결과 하단에는 전체 조합 일괄 저장 버튼 배치

기타 처리
- 결과 바로보기 버튼 → 애니메이션 스킵 후 즉시 결과 출력
- 속도변경 버튼 → 1x → 4x → 8x 순환
- 사운드 토글 버튼 → 효과음 on/off 처리
- 저장된 번호 localStorage 저장 / 중복 저장 방지
- 당첨확인 모달 → fetch + DOMParser 방식 동행복권 사이트 최신 당첨번호 크롤링 후 비교 처리
- cors 문제로 프록시 필수. 일치한 번호는 강조
- 모바일 반응형 필수
- HTML5 / ES6 표준 준수
- 코드 주석 충분히 작성
- 모듈화 금지 → 단일 파일 구성 유지
- 기능 추가 금지 / UX 변경 금지 / 스타일 임의 변경 금지

  • 저장소 Settings 탭 클릭
  • 좌측 메뉴에서 'Pages' 선택
  • 'Branch' 섹션에서 'main' 선택
  • '/(root)' 폴더 선택 후 저장
  • 몇 분 후 배포 완료 확인

NFC 태그에 URL 쓰기

  1. NFC 태그 쓰기 앱 설치 및 실행
  2. '새 기록' 또는 'URL' 옵션 선택
  3. 웹사이트 URL 입력 (예: https://username.github.io/my-website)
  4. '쓰기' 버튼 클릭
  5. 스마트폰을 NFC 태그에 가까이 대기
  6. 쓰기 완료 확인

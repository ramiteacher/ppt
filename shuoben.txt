• # AI로 나만의 웹사이트 만들기 - 최종 발표 대본

## 슬라이드 1: 표지

• (미소 지으며 천천히)
• 안녕하세요~ 반갑습니다 여러분. 오늘 여러분과 이런 자리에 함께하게 된 크리에이티브크루 한종윤입니다.
• 오늘 "AI로 나만의 웹사이트 만들기"라는 주제로 이야기를 나눌 건데요, 특히 비전공자 분들도 쉽게 따라하실 수 있는 내용으로 준비했습니다.
• 아마 여기 계신 분들 대부분 "웹사이트? 그건 개발자들이나 만드는 거지...", "나는 그런 거 못해..." 이런 생각 한번쯤 해보셨을 겁니다.

## 슬라이드 2: 왜 시작했나?

• 맞아요. 불과 네달전만 해도 저도 그랬거든요. 세상이 진짜 빠르게 변하고 있습니다.
• 저는 원래 크리에이티브 크루와 같은 건물에 있는 헬스장에서 상담과 영상편집 등 모든일을 다하던 소위 잡부 FC였어요. 2년 반 동안 근무했지만 적성에 맞지 않아 같은 건물 지하 3층에 위치한 크리에이티브 크루의 영상 편집 분야로 면접을 보게 되었고, 그때 김진산 대표님과 함께 하게 되었습니다. (감사합니다)
• 입사 후 처음 맡은 업무는 숏폼 영상들의 데이터를 수집하는 일이었는데요, 당시 아무것도 몰랐던 저는 자동화해야겠다는 생각으로 AI에게 여러 질문을 해보았고, 그대로 따라가다 보니 자동화 툴이 완성되더라고요?
• 이후에도 프로그램이나 웹사이트를 만들며, AI의 기술력으로 누구나 만들 수 있다는 큰 깨달음을 얻었습니다. 인간의 언어로 컴퓨터와 대화하는 일은 정말 흥미롭고 큰 성취감을 주었어요. 불과 입사 3달 만에 이 모든 일이 가능했습니다.
• 이렇듯 비전공자도 쉽게 AI로 재미 볼 수 있다는 것이죠~ 꿀은 선착순입니다!

## 슬라이드 3: 로드맵

• 그럼 오늘 저와 함께 어떤 이야기를 나눌지 간단히 정리해볼게요.
• (슬라이드를 가리키며) 보시는 것처럼 오늘의 목표는 "코딩 지식 없이도 AI의 도움으로 웹사이트 제작 및 배포하기"입니다. 정말 멋지지 않나요?
• 첫 번째로, 당장 시작해야 하는 이유에 대해 이야기해볼게요. 이건 단순한 기술 이야기가 아니라 여러분의 비즈니스, 여러분의 브랜드를 한 단계 업그레이드할 수 있는 이야기거든요.
• 두 번째, 필요한 툴들은 무엇인지 알아볼게요. 준비물이 많지 않아요. 컴퓨터와 몇 가지 프로그램만 있으면 됩니다.
• 세 번째, 예시 프로젝트를 함께 만들어볼 거예요.
• 네 번째, 무료로 배포하는 방법을 알아볼게요. 만든 웹사이트를 인터넷에 올려서 누구나 접속할 수 있게 만드는 방법이죠.
• 마지막으로는 추가 팁들과 질의응답으로 마무리하겠습니다!
• 50분이라는 시간이 짧게 느껴질 정도로 알찬 내용으로 준비했습니다. 함께 즐겁게 배워봐요!

## 슬라이드 4: 온라인 셀러들의 고민

• 요즘 온라인 셀러 분들 진짜 많잖아요? 스마트스토어, 쿠팡, 네이버 쇼핑... 다들 온라인으로 장사 한번 해볼까? 이런 생각은 누구나 합니다.
• 문제는 뭐냐? 경쟁이 너무 치열하다는 거죠. 똑같은 제품을 팔면 결국 가격 경쟁밖에 안 됩니다. 마진은 계속 줄어들고, 광고비는 계속 올라가고... 이런 악순환에 빠지게 되죠.
• 그래서 많은 분들이 "나만의 상품을 만들어 보고 싶은데...", "어떤 걸 만들어볼까?", "개인 브랜딩 하고 싶은데...", "나만의 서비스 이런 거 없을까?" 이런 고민을 하세요.
• 결국 대부분 "아 몰라 그냥 도매해와서 팔자.", "하던 방식대로 하자." 이렇게 되죠.
• 이게 진짜 한계예요. 브랜드 이미지도 없고, 확장성도 없어요. 결국 경쟁에서 살아남기 힘들어집니다.
• 지금 시대는 단순히 물건만 파는 시대가 아니에요. 경험을 팔고, 스토리를 팔고, 브랜드를 파는 시대입니다.

## 슬라이드 5: 실제 사례 - 야구 유니폼 NFC 키링

• 그럼 제가 실제로 진행했던 재미있는 사례 하나 말씀드릴게요.
• 크랩스 회원님 중 한 분께서 저에게 이런 요청을 주셨어요. "야구 유니폼 모양의 NFC 키링을 만들 건데요, 스마트폰으로 키링을 태그하면 각 팀 경기 하이라이트나 최근 경기 결과, 실시간 순위 같은 걸 바로 보여주고 싶어요."
• (슬라이드의 이미지를 가리키며) 보시는 것처럼 정말 다양한 기능이 필요했어요. 실시간 경기 결과, 팀 순위, 선수 정보, 경기 일정까지... 이런 걸 개발자한테 맡기면 돈 많이 깨질거예요.
• 결국 어떻게 했냐면요? 웹 디자인과 필요한 기능을 정리하고 HTML이랑 데이터 연결해서 GitHub Pages로 바로 배포해드렸어요. NFC 키링에는 그 링크 넣어버리면 끝나요.
• (링크를 클릭하는 제스처를 취하며) 이렇게 간단하게 태그만 하면 바로 정보가 뜨는 거죠.
• 물론 디자인은 정말 어렵습니다. 만약 직접 하신다면 디자인에 많은 시간을 쓰실 거예요. 하지만 AI의 도움을 받으면 이런 부분도 훨씬 수월해집니다.
• "이걸 내가 만들 수 있다고??" 하실 텐데, 네! AI의 도움만 있으면 가능합니다. 여기 계신 여러분도 조금만 배우면 충분히 가능해요.

## 슬라이드 6: 활용 가능성

• 여기서 더 흥미로운 건... AI를 활용하면 진짜 별별 걸 다 만들 수 있어요.
• (슬라이드의 각 예시를 가리키며)
- 로또 번호 추첨 사이트: 오늘 우리가 만들어볼 프로젝트죠. 재미있고 실용적이에요.
- 마법의 소라고둥: 고민이 있을 때 대답해주는 재미있는 사이트예요.
- 음식 메뉴 룰렛: "오늘 뭐 먹지?" 고민될 때 돌려보세요.
- 개인용 페이지: 웹사이트를 검색되게 설정하고 광고 삽입하면 부수입까지 생겨나요.
• 이런 거... 어려워 보이시죠? 하지만 여러분도 충분히 할 수 있습니다! 아이디어만 있다면 AI가 나머지는 다 도와줄 거예요.
• 이런 웹사이트들은 단순한 재미를 넘어서 여러분의 비즈니스에 실질적인 가치를 더해줄 수 있어요. 고객들에게 특별한 경험을 제공하고, 여러분의 브랜드를 기억에 남게 만들죠.

## 슬라이드 7: 준비물

• 그럼 뭘로 만들 수 있는지 준비물 간단히 정리해볼게요.
• (슬라이드의 각 아이콘을 가리키며)
- 첫째, 컴퓨터가 필요합니다. Windows, Mac, Linux 어떤 운영체제든 상관없어요.
- 둘째, VS Code라는 프로그램이 필요해요. 이건 무료로 다운로드 받을 수 있는 코드 에디터인데, 나중에 설치 방법 알려드릴게요.
- 셋째, GitHub 계정이 필요합니다. 이것도 무료로 가입할 수 있어요.
- 넷째, GitHub Copilot이라는 AI 도구가 필요한데, 이건 무료로 이용할 수 있어요. 다만 한도를 초과하면 월 $10 정도 하는 유료 서비스로 전환됩니다.
• (하단의 박스를 가리키며) 좋은 소식이 있어요! 학생이나 교육자 분들은 GitHub Education을 통해 무료로 사용할 수 있어요. 대학생이나 선생님이신 분들은 꼭 확인해보세요.
• 오른쪽에 보시면, NFC 키링을 만들고 싶으신 분들은 NFC 태그가 내장된 키링과 NFC 태그 쓰기 앱이 필요해요. 이건 선택사항이니까 관심 있으신 분들만 준비하시면 됩니다.

## 슬라이드 8: VS Code와 코파일럿 에이전트

• 우리의 핵심 도구인 VS Code와 코파일럿 에이전트에 대해 알아볼게요.
• (왼쪽 박스를 가리키며) 먼저 Visual Studio Code, 줄여서 VS Code라고 하는데요. 이건 무료로 사용할 수 있는 코드 에디터예요.
• 무료인데도 정말 강력한 기능들이 많아요. 확장 프로그램을 통해 기능을 계속 추가할 수 있고, 모든 운영체제에서 사용 가능해요. 게다가 가볍고 빠르게 실행되기 때문에 컴퓨터 사양이 좋지 않아도 부담 없이 사용할 수 있습니다.
• (오른쪽 박스를 가리키며) 오늘의 주인공, 코파일럿 에이전트입니다. 이건 AI 비서라고 생각하시면 돼요.
• 가장 놀라운 점은 자연어로 코드 생성을 요청할 수 있다는 거예요. "로또 번호 추첨 버튼 만들어줘"라고 말하면 정말로 그 기능을 하는 코드를 만들어줍니다. 복잡한 웹사이트도 자동으로 제작해주고요.
• 정말 ! 코딩 지식이 전혀 없어도 사용 가능해요. 여러분이 원하는 걸 일상 언어로 설명만 하면 됩니다.
• (하단의 강조 텍스트를 가리키며) 이게 바로 오늘의 핵심이에요. AI가 모든 코드를 자동으로 생성해준다는 거죠. 여러분은 아이디어만 제공하면 됩니다.

## 슬라이드 9: VS Code 설치 및 설정

• 실제로 VS Code를 설치하는 방법을 알아볼게요.
• 설치 과정은 정말 간단해요. 먼저 code.visualstudio.com 사이트에 접속하세요. 여기서 여러분의 운영체제에 맞는 버전을 다운로드하시면 됩니다.
• 운영체제에 맞는 설치 파일을 다운로드한 다음, 그 파일을 실행하고 안내에 따라 설치하시면 됩니다. 설치 과정에서 특별히 어려운 부분은 없어요. 그냥 '다음', '다음' 버튼을 클릭하시면 됩니다.
• 설치가 완료되면 확장프로그램에서 korean 검색하셔서 적용하면 한글화가 됩니다.
• (코파일럿 설정 부분을 가리키며) 코파일럿 설정도 간단합니다. VS Code 좌측 확장 아이콘을 클릭하고 'GitHub Copilot'을 검색해서 설치하세요. 'GitHub Copilot Chat' 확장도 설치해주세요. GitHub 계정 연결은 우측 하단 계정 아이콘을 클릭하면 됩니다. 에이전트 모드로 전환하려면 Ctrl+Shift+I 단축키를 사용하세요.

## 슬라이드 10: 오늘의 프로젝트: 로또 번호 추첨기

• 오늘 우리가 만들 로또 번호 추첨기에 대해 좀 더 자세히 알아볼게요.
• (중앙의 이미지를 가리키며) 이렇게 생긴 웹사이트를 만들 거예요.
• (왼쪽 박스를 가리키며) 주요 기능을 정리해볼게요.
- 로또 번호 자동 추첨 기능이 있어요. 1부터 45까지의 숫자 중에서 무작위로 6개를 뽑아줍니다.
- 단일 조합 모드와 5조합 모드를 선택할 수 있어요.
- 물리 엔진 기반으로 공이 움직이는 애니메이션이 있어서 실제 로또 추첨기처럼 생생한 느낌을 줍니다.
- 번호별로 색상이 다르게 표시되어 있어서 시각적으로 구분이 잘 되고요.
- 로컬 스토리지에 번호를 저장할 수 있어서 마음에 드는 번호는 따로 보관할 수 있습니다.
• (오른쪽 박스를 가리키며) 이 프로젝트의 특별한 점은 뭐냐면, 실제로 사용할 수 있는 웹사이트라는 거예요. 그냥 예제로 끝나는 게 아니라, 만들고 나면 진짜 써볼 수 있어요.
• 모바일에서도 잘 작동하고, 디자인도 깔끔하고 직관적이에요. 무엇보다, 이 모든 걸 AI의 도움을 받아 코딩 지식 없이도 만들 수 있다는 게 

## 슬라이드 11: 코파일럿 에이전트 사용하기

• 코파일럿 에이전트를 어떻게 사용하는지 알아볼게요.
• (슬라이드의 이미지를 가리키며) 코파일럿 에이전트는 자연어로 코드 생성 요청이 가능한 인터페이스를 제공합니다.
• 에이전트 실행 방법은 간단해요. VS Code 실행 및 새 폴더 생성 후, 하단 코파일럿 아이콘을 클릭하거나 Ctrl+Shift+I 단축키를 누르면 됩니다. 그러면 모든 드롭다운에서 'Agent' 선택이 가능해집니다.
• 프롬프트 입력 후 전송(Enter)하면 AI가 자동으로 파일 생성 및 코드 작성을 시작합니다. 정말 신기하죠?
• (왼쪽 박스를 가리키며) 에이전트 실행 방법을 정리해볼게요.
• 1. VS Code 실행 및 새 폴더 생성
• 2. 하단 코파일럿 아이콘 클릭 또는 Ctrl+Shift+I 단축키
• 3. 모든 드롭다운에서 'Agent' 선택
• 4. 프롬프트 입력 후 전송(Enter)
• 5. AI가 자동으로 파일 생성 및 코드 작성
• (오른쪽 박스를 가리키며) 에이전트 작동 방식도 살펴볼까요?
- 프롬프트 분석 및 계획 수립
- 필요한 파일 자동 생성(HTML, CSS, JS)
- 코드 작성 및 수정(실시간)
- 필요시 디버깅 명령 실행
- 오류 감지 및 수정(자동 디버깅)
- 코드 설명 및 추가 요청 대응
• 코딩 지식 없이도 프롬프트만으로 완성된 웹사이트 제작이 가능합니다!

## 슬라이드 12: 효과적인 프롬프트 작성법

• 정말 중요한 부분이에요. AI에게 어떻게 지시를 내려야 원하는 결과물을 얻을 수 있을까요?
• 프롬프트... 그냥 아무렇게나 쓰면 안 돼요. 좋은 프롬프트를 작성하는 방법을 알려드릴게요.
• 프롬프트 작성 5원칙을 기억하세요.
• 첫째, 구체적으로 작성하세요. "웹사이트 만들어줘"보다는 "로또 번호 6개를 무작위로 생성하고 결과를 화면에 표시하는 웹사이트를 만들어줘"라고 하는 게 훨씬 좋아요.
• 둘째, 참고 사이트 링크를 제공하세요. 비슷한 디자인이나 기능을 가진 사이트가 있다면 그 링크를 AI에게 알려주세요.
• 셋째, 파일 구조를 명시하세요. "index.html, style.css, script.js 세 개의 파일로 만들어줘"라고 구체적으로 말해주면 AI가 더 체계적으로 코드를 생성해줍니다.
• 넷째, 필수 기능을 정리하세요. 어떤 기능이 꼭 필요한지 목록으로 만들어서 알려주세요.
• 다섯째, 세부 스타일을 지정하세요. 색상, 폰트, 레이아웃 등 디자인적인 요소도 구체적으로 말해주면 더 좋아요.
• 이렇게 해줘야 AI가 진짜 원하는 걸 만들어줘요. 프롬프트가 명확할수록 결과물도 더 만족스러울 거예요.

## 슬라이드 15: GitHub Pages로 웹사이트 배포하기

• 웹사이트를 다 만들었다고 가정해볼게요. 그럼 이걸 어떻게 세상에 공개하냐? 이게 또 하나의 큰 장벽이었는데, 이제는 그렇지 않아요.
• 답은 GitHub Pages! 완전 무료로 웹사이트를 배포할 수 있는 서비스예요.
• (슬라이드의 이미지를 가리키며) 이렇게 생긴 GitHub Pages 설정 화면을 통해 여러분의 웹사이트를 인터넷에 올릴 수 있어요.
• (왼쪽 박스를 가리키며) GitHub 저장소 생성 과정은 정말 간단해요.
• 1. github.com/new에 접속해서 저장소 이름을 입력하고 'Public'을 선택한 다음 저장소를 생성하세요.
• 2. 'Public' 선택 후 저장소 생성을 하고, VS Code에서 생성한 파일들을 업로드하세요.
• 3. commit 및 push로 코드를 저장하면 됩니다.
• (오른쪽 박스를 가리키며) GitHub Pages 활성화 과정도 간단해요.
• 1. 저장소의 Settings 탭을 클릭하고, 좌측 메뉴에서 'Pages'를 선택합니다.
• 2. 'Branch' 섹션에서 'main'을 선택하고, '/(root)' 폴더를 선택한 후 저장하세요.
• 3. 몇 분 후에 배포가 완료되면, 여러분만의 URL이 생성됩니다.
• 몇 분 후 https://[사용자이름].github.io/[저장소이름]/에서 웹사이트를 확인할 수 있습니다!

## 슬라이드 16: NFC 태그에 웹사이트 연결하기

• NFC 태그를 활용하면 물리적 제품과 디지털 콘텐츠를 연결할 수 있어요.
• 준비물로는 NFC 기능이 있는 스마트폰, NFC 태그, NFC 태그 쓰기 앱이 필요합니다.
• NFC 태그에 URL 쓰기 방법은 간단해요:
• 1. NFC 태그 쓰기 앱 설치 및 실행
• 2. '새 기록' 또는 'URL' 옵션 선택
• 3. 웹사이트 URL 입력 (예: https://username.github.io/my-website)
• 4. '쓰기' 버튼 클릭
• 5. 스마트폰을 NFC 태그에 가까이 대기
• 6. 쓰기 완료 확인
• 팁: URL을 짧게 만들려면 URL 단축 서비스(예: Bitly)를 사용하면 좋습니다.

## 슬라이드 17: 온라인 셀러를 위한 비즈니스 활용 아이디어

• 우리가 배운 기술을 실제 비즈니스에 어떻게 활용할 수 있는지 아이디어를 나눠볼게요.
• 첫 번째 아이디어는 주문 제작 페이지입니다. 만약 내가 팔찌를 DIY하여 파는 판매자라면, 고객이 직접 비즈와 끈을 골라 하나하나 채워서 구매할 수 있는 페이지를 만들 수 있어요. 고객 참여형으로 사이트를 만들면 할 수 있는 주문제작 아이템이 많아지겠죠?
• 두 번째 아이디어는 선물 페이지입니다. 주로 선물되는 상품들, 예를 들어 인형이나 작은 인테리어 소품 등에 구매자가 선물하고 싶은 사람에게 메시지를 남길 수 있게 하는 거예요. 상품에 태그하면 선물한 사람의 편지나 영상들을 띄워주는 이런 귀여운 상품들! 구매자가 구매할 때 판매자에게 요청하는 대로 우린 정해진 템플릿에 내용만 바꿔서 보내주면 돼요!
• 세 번째 아이디어는 웹페이지를 검색엔진에 등록하고 광고를 삽입하여 부수입을 만드는 것입니다. 대신 컨텐츠의 양과 질로 승부를 봐야하니 꾸준한 업데이트가 필요합니다!
• 커스텀 제품으로 높은 마진을 얻을 수 있고, 경쟁업체와의 차별점을 제공하고, 고객들에게 특별한 경험을 선사하는 거죠.
• 또한, 동일한 NFC 태그에 새로운 콘텐츠를 업데이트하여 고객 재방문을 유도할 수 있어요. 웹사이트 내용만 바꾸면 되니까 물리적인 제품을 다시 만들 필요가 없어요.

## 슬라이드 18: 추가 팁과 트릭

• AI와 더 효과적으로 협업하기 위한 몇 가지 팁을 알려드릴게요.
• 첫째, 참고한 웹사이트 주소를 제공하세요.
• 둘째, 복잡한 요청은 단계적으로 나눠서 요청하세요. 한 번에 너무 많은 것을 요구하면 AI가 혼란스러워할 수 있어요.
• 셋째, 피드백을 자주 주세요. "이 부분을 더 크게 만들어줘", "이 색상을 좀 더 밝게 해줘" 같은 구체적인 피드백을 주면 결과물이 점점 더 좋아질 거예요.
• 넷째, 코드 설명을 요청하세요. AI가 생성한 코드가 어떻게 작동하는지 설명해달라고 하면, 여러분도 조금씩 코딩에 대해 배울 수 있어요.

• 몇 가지 더 알아두면 좋은 팁이 있어요:
- 로고 변경: "로고를 내 브랜드 이미지로 바꿔줘"
- 색상 테마: "primary-color를 #FF5733으로 변경해줘"
- 폰트 변경: "글꼴을 'Noto Sans KR'로 바꿔줘"
- 기능 추가: "연락처 폼을 추가해줘"
• 이런 팁들을 활용하면 AI와의 협업이 훨씬 더 효율적이고 즐거워질 거예요.

## 슬라이드 19: 자주 묻는 질문

• 여러분이 가질 수 있는 몇 가지 질문들에 미리 답변해 드릴게요.
Q. 코딩 지식이 전혀 없어도 가능한가요?
A. 네, 프롬프트만 잘 작성하면 웹사이트 제작이 가능합니다. 코딩 지식이 전혀 없어도 됩니다.
Q. 다른 호스팅 방법은 없나요?
A. Netlify, Vercel, Firebase 등 무료 호스팅 서비스를 이용할 수 있습니다. 이들도 무료로 시작할 수 있어요.
Q. 저작권 문제는 없나요?
A. 코파일럿으로 생성한 코드는 상업적으로 사용 가능합니다. 다만, 다른 사이트의 디자인을 그대로 복제하는 것은 피하는 게 좋아요.
Q. 웹사이트 수정은 어떻게 하나요?
A. VS Code에서 파일을 열고 AI에게 "이 부분을 이렇게 수정해줘"라고 요청하면 됩니다. 수정한 후에는 다시 GitHub에 업로드하면 웹사이트가 자동으로 업데이트됩니다.
• 코파일럿 에이전트는 코딩 관련 모든 질문에 답변할 수 있는 AI 비서입니다. 궁금한 점이 있으면 언제든지 물어보세요!
